## Prerequisites:
- Node and NPM
- PHP
- Composer
- RabbitMQ default installation;

## Demo
To run the app do the following in project root:
```
# start app's backend
npm install

composer update

node index.js

php socket.php

# now start the app frontend
cd frontend

npm install

# serve with hot reload at localhost:8080
npm run dev

```

In your browser go to url `http://localhost:8080`
